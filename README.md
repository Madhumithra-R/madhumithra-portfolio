# 🌐 Madhumithra R — Portfolio

> **Live Site:** [madhumithra-portfolio.vercel.app](https://madhumithra-portfolio.vercel.app)

A modern, responsive personal portfolio website built to showcase my journey as a Computer Science student passionate about Web Technologies and SAP/ERP Solutions.

---

## ✨ Features

- 🎯 **Animated Particle Background** — Cyan floating dots with connecting lines
- 📱 **Fully Responsive** — Mobile, Tablet & Desktop
- 🌙 **Dark Theme** — Deep black with cyan accents
- ⚡ **Smooth Scroll Animations** — Reveal on scroll
- 📄 **Resume Download** — One-click PDF download
- 📬 **Contact Form** — With real-time validation

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/Madhumithra-R/madhumithra-portfolio.git

# Navigate to project
cd madhumithra-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

---

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Deployed on Vercel — auto-deploys on git push ✅
```
